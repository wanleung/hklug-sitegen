國際IT匯 2014 - 自由及開源軟件在香港研討會

By sammyfung (Open Source Hong Kong) · 2014-03-31 05:36:03

News Feed - Source :
Open Source Hong Kong - 國際IT匯 2014 - 自由及開源軟件在香港研討會

在香港，不論是個人或團體，都有不少透過使用自由及開源軟件獲益的故事。藉著今次「國際IT匯」，香港Linux用家協會請了多位資訊科技專業人士，分享他們參與自由及開源軟件的經驗。

日期: 2014年4月12日, 星期六
時間: 下午2:00-5:30.
地點: 香港城市大學一號教學樓 LT3
網上登記: https://hklugitfest2014.eventbrite.com/
網站: http://opensource.hk/en/itfest-2014
語言: 英語或廣東話 (英文slides)
查詢: webmaster@opensource.hk

議程

2:00pm 接待
2:15pm Open Source Marketing (Sammy Fung) (30分鐘)
2:50pm Open Source and Startup (Wan Leung Wong) (30分鐘)
3:20pm 小休及閃電講
3:35pm Use of Open Source OpenStack Cloud Platform (Dick Tang) (30分鐘)
4:10pm Data-HK Cloud based Data Platform 香港數據 - 雲端數據平台 (Cyrus Wong, Ron Tam) (30分鐘)
4:40pm 小休
4:50pm Switching from MySQL to PostgesSQL - Part 2 (Triton Ho) (40分鐘)
5:30pm 完結

展覽
