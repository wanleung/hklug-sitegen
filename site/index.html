會務: 正構思重開monthly event, 現正仍在物色適合場地

Written by HKLUG on 2018-03-06 19:16.

會務: 正構思重開monthly event, 現正仍在物色適合場地.

重開之event為Hacking Saturday.

計劃, 有前菜同主菜, 由下午開始, 可能2:30/3:00pm開始, 至6:00pm, 此時段以TALK形式講解, 如無主要TOPIC有人想講, 就會轉為吹水討論. 之後就是食飯休息或散水. 而主菜, 會是晚上大約8時或9時開始, 以hacking為主, 專FOR夜鬼, 若場地許可, 是會通頂的.

而hacking主題, 係會以我們想玩, 但又無什麼時間玩的PROJECT為主. 參加者要求, 自備食物酒水及設備. 部份設備有可能我們能提供, 但建議自己都買一套. 下午部份, 歡迎各界人士參與.

HACKING部份, 暫以業界人士為主. 新手初學者都可以參加, 但未必能教到所需SKILL SET而跟不上, 此需要留意.

現今有的PROJECT或會玩的野如下:

  1. Henry Law 塊印度開源VIDEO BOARDCAST板. 買左成年重未有人整郁到.

  2. Firewall, VPN, Tor node 等實作, 整合.

  3. 日本人Niibe 支FST-01 加密手指開發使用同整合. (買左一堆無時間玩)

  4. Blockchain - coin, smart contert, application

  5. Blockchain - pure chain technology

  6. Robotics: Servo controller board linux support, building sdk for linux

  7. 3D printing related projects, including print server, print protocol, 3D modeling tools and so on.

  8. By Ben Lau, 熱感紙的mini printer

  9. Opencv
