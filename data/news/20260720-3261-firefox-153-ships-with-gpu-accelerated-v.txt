Title: Firefox 153 Ships with GPU-Accelerated Video and a New Enterprise Branch
Content:
<p>Mozilla has released Firefox 153, a dual-focused update that introduces GPU-powered video decoding for enhanced performance while launching a new long-term support channel for enterprise stability. The monthly update brings the first integration of Vulkan Video decoding, a fresh Extended Support Release (ESR) version, and the cautious, disabled-by-default introduction of JPEG-XL image format support.</p>
<p>The headline performance advancement is Vulkan Video decoding on Windows and Linux. This feature offloads video processing from the CPU to the GPU via the Vulkan API. For users with compatible graphics hardware, this should result in smoother playback of high-resolution and high-framerate video, lower power consumption, and improved system responsiveness during media-intensive browsing.</p>
<p>Concurrently, Firefox 153 inaugurates the ESR 153 branch. This version is designed for organizational IT administrators, offering a stable foundation that receives only critical security patches over an extended support cycle, typically a year. ESR 153 provides a predictable deployment target, allowing businesses to balance security updates with application compatibility in a managed environment.</p>
<p>Mozilla is also proceeding carefully with the next-generation JPEG-XL image format. The capability is included in this release but remains experimental and must be manually enabled. This staged approach suggests Mozilla is assessing the standard's real-world utility and ecosystem readiness before committing to default support.</p>
<p>Firefox 153 therefore delivers on two distinct fronts: pushing the boundaries of browser performance for end-users and developers through modern GPU acceleration, and offering a steadfast, security-focused platform for institutional needs. The release is available now for download on all supported platforms.</p>
<hr>
<p>Mozilla 推出 Firefox 153，這是一項雙管齊下的更新，引入由 GPU 驅動的視頻解碼功能以提升效能，同時推出新的長期支援渠道，為企業用戶提供穩定性。每月更新帶來首次整合 Vulkan 視頻解碼、全新的 Extended Support Release（ESR）版本，以及謹慎地以預設停用方式初步引入 JPEG-XL 圖像格式支援。</p>
<p>核心效能提升在於 Windows 和 Linux 平台上的 Vulkan 視頻解碼。此功能透過 Vulkan API 將視頻處理工作從 CPU 轉移到 GPU。對於使用相容硬件的用戶，這應能帶來更流暢的高分辨率及高幀率視頻播放效果、更低的功耗，並在進行媒體密集型瀏覽時提升系統回應速度。</p>
<p>與此同時，Firefox 153 正式啟用 ESR 153 分支。此版本專為機構 IT 管理員設計，提供穩定基礎，在通常為一年的擴展支援週期內僅接收關鍵安全補丁。ESR 153 提供可預測的部署目標，讓企業能在受控環境中平衡安全更新與應用程式相容性。</p>
<p>Mozilla 亦謹慎推進新一代 JPEG-XL 圖像格式。此功能已包含在本版本中，但仍屬實驗性質並需手動啟用。這種分階段推出的方式表明，Mozilla 正在評估該標準的實際效用及生態系統就緒程度，之後才會將其設為預設支援。</p>
<p>因此，Firefox 153 在兩方面同步發力：一方面透過現代 GPU 加速技術，為終端用戶及開發者拓展瀏覽器效能的極限；另一方面為機構需求提供穩定、以安全為本的平台。此版本現已開放，適用於所有支援平台下載。</p>
