Title: Canonical Brings Day-One Ubuntu Support to Microsoft's Second-Gen Azure Cobalt 200 Arm VMs
Content:
<p>Microsoft has launched the preview of Azure Cobalt 200, its second-generation custom Arm-based processor, with Canonical announcing immediate availability of both Ubuntu and Ubuntu Pro on the new virtual machines. The move underscores a deepening strategic alignment between the two companies aimed at making Arm-based cloud computing a frictionless experience for enterprise workloads.</p>
<h4>No Engineering Changes Required</h4>
<p>According to Canonical, organisations can deploy Ubuntu on the new Azure Cobalt 200 instances without any modifications to their existing applications or platform configurations. This is a significant detail, as the need for software porting and re-engineering has historically been one of the most cited barriers preventing enterprises from moving workloads to Arm-based cloud infrastructure.</p>
<p>The Cobalt 200 represents Microsoft's continued investment in custom silicon for its Azure platform. The new processors are designed to deliver improved performance-per-watt for general-purpose and scale-out workloads, competing in a market segment where Arm-based designs have steadily gained traction against traditional x86 architectures in cloud data centres.</p>
<h4>Ubuntu Pro Targets Production Deployments</h4>
<p>For organisations planning to run production workloads on the new instances, Canonical is steering users toward Ubuntu Pro, its commercial subscription offering. The pitch centres on two capabilities particularly relevant to cloud operations: Kernel Livepatch and extended security maintenance (ESM).</p>
<p>Kernel Livepatch allows critical kernel security patches to be applied without rebooting the system — a feature that directly addresses uptime requirements for always-on production services. Extended security maintenance, meanwhile, provides security updates for Ubuntu packages beyond the standard support window, giving enterprises a longer runway before they need to upgrade their base operating system.</p>
<p>These features collectively position Ubuntu Pro as Canonical's recommended path for any serious deployment on the Cobalt 200 platform, rather than the freely available Ubuntu images.</p>
<h4>Broader Implications for the Arm Ecosystem</h4>
<p>The day-one availability of Ubuntu on a major new Arm chip launch from Microsoft signals how far the Arm server ecosystem has matured. A few years ago, running a mainstream Linux distribution on a new Arm platform often required community effort and weeks of post-launch enablement work. The coordinated launch between Canonical and Microsoft suggests that Arm compatibility is now treated as a first-class priority by both cloud providers and Linux vendors alike.</p>
<p>For IT teams evaluating their cloud strategies, the combination of custom Arm silicon with a well-supported Linux distribution lowers the barrier to experimentation. Organisations that have been watching the Arm cloud space from the sidelines may find the Cobalt 200 preview a low-risk opportunity to benchmark their workloads against traditional x86 instances.</p>
<p>The Azure Cobalt 200 preview is available now, with Ubuntu images accessible through the Azure Marketplace.</p>
<hr>
<p>Microsoft 已推出其第二代訂製 Arm 處理器 Azure Cobalt 200 的預覽版，與此同時，Canonical 宣佈 Ubuntu 和 Ubuntu Pro 將可即時在這些新虛擬機上使用。此舉凸顯了兩家公司之間日益深化的戰略合作，旨在為企業工作負載提供無縫的 Arm 雲端運算體驗。</p>
<h4>無需工程修改</h4>
<p>據 Canonical 表示，企業組織無需對現有應用程式或平台配置進行任何修改，即可在新的 Azure Cobalt 200 實例上部署 Ubuntu。這是一個關鍵細節，因為歷史上，軟件移植和重新工程化的需要一直是阻礙企業將工作負載遷移至 Arm 雲端基礎設施的主要障礙之一。</p>
<p>Cobalt 200 代表了 Microsoft 在為其 Azure 平台訂製晶片方面的持續投資。新款處理器旨在為通用和橫向擴展工作負載提供更佳的每瓦性能，在 Arm 架構於雲端數據中心穩步蠶食傳統 x86 架構市場份額的背景下參與競爭。</p>
<h4>Ubuntu Pro 瞄準生產部署</h4>
<p>對於計劃在新實例上運行生產工作負載的企業組織，Canonical 引導用戶採用其商業訂閱服務 Ubuntu Pro。其推廣重點集中在兩項與雲端營運特別相關的功能：Kernel Livepatch 與延長安全維護服務（ESM）。</p>
<p>Kernel Livepatch 允許在無需重新啟動系統的情況下應用關鍵的核心安全修補程式——這項功能直接解決了全天候運行的生產服務對運行時間的需求。而延長安全維護服務則在標準支援窗口之外，為 Ubuntu 套件提供安全更新，給予企業更充裕的時間來升級其基礎作業系統。</p>
<p>這些功能共同將 Ubuntu Pro 定位為在 Cobalt 200 平台上進行任何正式部署時 Canonical 推薦的方案，而非免費提供的 Ubuntu 映像。</p>
<h4>對 Arm 生態系統的更廣泛影響</h4>
<p>Ubuntu 在 Microsoft 重要新款 Arm 晶片發佈時首日即可用，這顯示了 Arm 伺服器生態系統已發展得何等成熟。幾年前，在一個新的 Arm 平台上運行主流的 Linux 發行版，往往需要社區的努力和發佈後數週的啟用工作。Canonical 與 Microsoft 的協調發佈表明，Arm 相容性現已被雲端供應商和 Linux 供應商視為首要優先事項。</p>
<p>對於正在評估其雲端戰略的 IT 團隊而言，結合訂製 Arm 晶片與一個支援完善的 Linux 發行版，降低了實驗的門檻。那些一直對 Arm 雲端領域持觀望態度的企業組織，可能會發現 Cobalt 200 預覽版是一個低風險的機會，可用來對其工作負載與傳統 x86 實例進行基準測試。</p>
<p>Azure Cobalt 200 預覽版現已可用，Ubuntu 映像可透過 Azure Marketplace 取得。</p>
